# Subject: Linear Algebra

## What this subject teaches
- A vector is an object with magnitude and direction; vectors live in a vector space defined by closure under addition and scalar multiplication.
- A linear map between vector spaces is uniquely represented by a matrix; applying the map is matrix-vector multiplication.
- Composing linear maps corresponds to matrix multiplication (non-commutative).
- An eigenvector of a square matrix is a nonzero vector that the matrix only stretches; the stretch factor is its eigenvalue.
- Diagonalization (A = P D P^-1) turns repeated application of a matrix into elementwise exponentiation along its eigenvectors.

## Prerequisites assumed
- High-school algebra (arithmetic, basic functions). No prior college math required.

## Key concepts (with one-line definitions)
- Vector: object with magnitude and direction; can be added and scaled.
- Vector space: a set closed under vector addition and scalar multiplication, with a zero element.
- Linear map: a function T satisfying T(u+v) = T(u)+T(v) and T(cv) = cT(v).
- Matrix: a rectangular array of numbers that encodes a linear map between finite-dimensional vector spaces.
- Eigenvalue / eigenvector: a scalar lambda and nonzero vector v with A v = lambda v.
- Diagonalization: factorization A = P D P^-1 where D is diagonal and the columns of P are eigenvectors of A.
