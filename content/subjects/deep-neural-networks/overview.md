# Deep Neural Networks

A deep neural network is a function-approximation machine built by stacking
many small, simple computational units. Despite their innocent-looking
ingredients, they can learn to recognize images, translate languages, fold
proteins, and beat world champions at Go.

This course will teach you the *minimum* you need to read modern machine
learning research and build a small network yourself. By the end you'll
understand:

- what a single perceptron is and why it isn't enough on its own
- how multi-layer networks compose nonlinearities to approximate functions
- how gradient descent and the backpropagation algorithm let networks learn
  from data

You will benefit from already knowing some linear algebra (vectors and
matrices) — it's listed as a prerequisite.
